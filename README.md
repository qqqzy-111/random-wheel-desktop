# 彩色随机转盘 - 项目说明

## 📋 项目简介

这是一个功能完整的 **Python 桌面应用程序**，用于从 Excel 文件读取数据生成彩色转盘，支持随机旋转抽奖、实时数据更新等功能。

**特点**：
- ✅ 完全离线运行，无需网络
- ✅ 支持动态加载 Excel 数据
- ✅ 流畅的旋转动画效果
- ✅ 可打包成单个 EXE 文件
- ✅ 支持 Windows 桌面快捷方式

---

## 🚀 快速开始（3种方式）

### 方式一：最简单（推荐首次使用）

1. **打开文件夹** → 双击 `run.bat`
2. **选择 Excel 文件** → 点击"📁 选择Excel文件"
3. **开始抽奖** → 点击"🎪 开始抽奖"

✅ **无需安装任何东西，一键运行！**

### 方式二：使用 Python 命令

```powershell
# 打开 PowerShell，进入文件夹
cd "d:\风吹哪页读哪页"

# 安装依赖（首次运行一次）
pip install -r requirements.txt

# 运行程序
python main.py
```

### 方式三：打包成桌面应用

```powershell
# 方法1：双击 build.bat 自动打包
build.bat

# 方法2：手动打包
pip install pyinstaller
pyinstaller -F -w main.py

# 然后用 create_shortcut.bat 创建快捷方式
create_shortcut.bat
```

之后可以像普通 Windows 应用一样使用，双击快捷方式即可打开。

## 📝 Excel 数据格式

### 标准格式

```
姓名          ← 第一行是表头（任意内容，会被跳过）
张三          ← 第二行开始是实际数据
李四
王五
赵六
孙七
```

### 规则

- ✅ 数据在**第一列**（A列）
- ✅ 第一行是表头（会自动跳过）
- ✅ 支持中文、英文、数字、符号
- ✅ 自动过滤空白行
- ❌ 不支持多列数据（只读第一列）
- ❌ 不支持合并单元格
- ❌ 不支持公式（只读显示值）

## 🔧 技术架构

### 技术栈
- **GUI框架**：Tkinter（Python内置）
- **绘图**：Canvas（Tkinter）
- **Excel处理**：openpyxl 库
- **动画**：threading + requestAnimationFrame风格的定时更新
- **数学计算**：math 库处理转盘几何

### 核心类

```python
WheelSpinner      # 转盘管理类
  ├── set_items()      # 设置选项
  ├── draw()           # 绘制转盘
  ├── spin()           # 启动旋转
  └── get_winner()     # 获取中奖者

ExcelReader       # Excel读取类
  └── read_first_column()  # 读取第一列

WheelApp          # 主应用类
  ├── _create_ui()           # 创建界面
  ├── _load_excel_data()     # 加载数据
  └── _on_start_btn_click()  # 处理事件
```

## 🛠️ 常见问题

### Q：可以在哪些Windows版本上运行？
A：Windows 7 及以上都支持（包括 Windows 10/11）

### Q：需要网络连接吗？
A：不需要，完全离线运行

### Q：EXE文件能分享给别人吗？
A：可以！直接把 `dist\main.exe` 发给别人，别人可以直接运行

### Q：能修改源代码吗？
A：可以！`main.py` 文件包含详细注释，修改后重新打包即可

### Q：安全吗？
A：完全安全。源代码开放，数据只在本机处理，不上传到任何服务器

### Q：多少条数据上限？
A：理论上无限制。实际上显示超过 100 条时，文字会比较小

### Q：能保存抽奖记录吗？
A：可以修改源代码添加这个功能。本版本不保存记录

## 📞 技术支持

### 遇到问题的解决步骤

1. **查看错误信息**
   - 如果出现错误，记下错误信息的前两行

2. **查阅文档**
   - 在 SETUP_GUIDE.md 中搜索相关关键词

3. **尝试重装依赖**
   ```powershell
   pip install --upgrade openpyxl
   ```

4. **重新运行**
   ```powershell
   python main.py
   ```

### 常见错误

| 错误信息 | 原因 | 解决方案 |
|---------|------|--------|
| ModuleNotFoundError: openpyxl | 缺少依赖 | `pip install openpyxl` |
| No such file or directory | Excel文件路径错误 | 检查文件是否存在 |
| TypeError: unsupported operand | Excel数据格式错误 | 检查第一列数据格式 |
