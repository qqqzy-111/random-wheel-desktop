【步骤1】运行程序
─────────────────

─────────────────────
点击程序中的"📁 选择 Excel 文件"按钮
选择你的 Excel 数据文件
（Excel 第一列放数据，第一行是表头）

【步骤3】开始抽奖
──────────────
点击"🎪 开始抽奖"按钮
转盘旋转后显示中奖者

========================================
🎯 三种使用方式
========================================

【方式一】开发模式（推荐新手）
└─ 双击 run.bat 直接运行
└─ 无需任何安装
└─ 最简单快速

【方式二】Python命令模式（推荐开发者）
└─ pip install -r requirements.txt
└─ python main.py
└─ 适合有 Python 基础的用户

【方式三】打包成 EXE（推荐分发）
└─ 双击 build.bat 自动打包
└─ 生成 dist/main.exe 可执行文件
└─ 可创建桌面快捷方式
└─ 像普通 Windows 软件一样使用

========================================
📦 打包成桌面应用的步骤
========================================

第1步：打包程序
   ├─ 方式A：双击 build.bat（推荐）
   └─ 方式B：命令行输入 pyinstaller -F -w main.py

第2步：找到 EXE 文件
   └─ 路径：d:\风吹哪页读哪页\dist\main.exe

第3步：创建快捷方式
   ├─ 方式A：右键 main.exe → 发送到 → 桌面
   └─ 方式B：双击 create_shortcut.bat（自动）

第4步：使用快捷方式
   └─ 在桌面找到快捷方式，双击打开

========================================
📝 Excel 数据格式
========================================

✅ 正确的格式：

┌─────────────┐
│ 学生名单    │  ← 第一行（表头）
├─────────────┤
│ 张三        │  ← 第二行开始是实际数据
│ 李四        │
│ 王五        │
│ 赵六        │
│ 孙七        │
└─────────────┘

✅ 要求：
  • 数据在第一列（A列）
  • 第一行是表头（会自动跳过）
  • 从第二行开始是实际数据
  • 支持任何格式的文本

❌ 不支持：
  • 多列数据（只读第一列）
  • 合并单元格
  • 公式（只读显示值）

========================================
💡 常见问题
========================================

Q: 双击 run.bat 后闪退了
A: 请查看 QUICK_START.md 的"问题1"部分

Q: 提示 Python not found
A: 需要安装 Python，请查看 SETUP_GUIDE.md

Q: Excel 文件无法读取
A: 检查数据是否在第一列，第二行开始

Q: 想修改程序的颜色或动画速度
A: 编辑 main.py 文件，参考代码中的注释

Q: EXE 文件有多大
A: 约 150-200 MB（包含完整 Python 解释器）
